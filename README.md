# Designer Portfolio

This project is a personal portfolio for a designer, showcasing their work, biography, CV, and contact information. The portfolio is structured to provide a seamless user experience with a clean and modern design.

## Project Structure

- **portfolio.html**: The main portfolio page that includes navigation, hero section, and project showcases.
- **bio.html**: The biography page that provides information about the designer's background and experience.
- **cv.html**: The CV page that outlines the designer's professional experience, education, and skills.
- **contact.html**: The contact page that allows users to reach out to the designer for potential collaborations.
- **style.css**: The stylesheet that defines the layout, typography, and visual elements of the portfolio.
- **script.js**: The JavaScript file that adds interactivity and functionality to the portfolio.

## Usage

To view the portfolio, open `portfolio.html` in a web browser. Use the navigation links to explore the different sections of the portfolio, including the biography, projects, CV, and contact information.
